    Samsung Galaxy ACE

    Project maintained by RafaelBaugis Hosted on GitHub Pages — Theme by mattgraham

    Model: GT-S5830C-i-M

    CPU: Broadcom 21553 ARM11

    Arquitecture: ARM1136JF-S

    This Kernel version 2.6.35.7 is from Samsung sources, with GCC 4.4.7 compiled from sources, specific to arm1136jf-s. I changed in sources for add more frequencies to cpu and configured to add features. The kernel can be used on any rom.

    kernel features:

    • vfat, ext2, ext3, ext4.
    • swap, ramzswap and zRAM.
    • cpu freq with auto undervolt.
      1. 156Mhz 1,18v
      2. 312Mhz 1,24v
      3. 624Mhz 1,30v
      4. 832Mhz 1,36v
    • cpu auto undervolt on change cpu freq.
    • cpu governors smartassV2, lazy, lulzactive, minmax, lagfree, lionheart, conservative, interactive, ondemand, ondemandx, userspace, powersave, bcm21553 and performance.
    • IO schedulers noop, deadline, cfq, sio, vr and bfq.
    • iptables full needed rules.
    • vpn support
    • cifs support
    • init.d - if not work check if the su and busybox location is /system/xbin/

    How to Compile:

    1. Install dependencies.
      • $ ./install_dep.sh
    2. Configure and Compile.
      • $ ./Compile.sh
    3. Create package.
      • $ ./Creat_package.sh

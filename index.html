Kernel cooperve

Samsung Galaxy ACE

Download Download View Kernel cooperve on GitHub

Model: GT-S5830C-i-M

CPU: Broadcom 21553 ARM11

Arquitecture: ARM1136JF-S

This Kernel version 2.6.35.7 is from Samsung sources, with GCC 4.4.7 compiled from sources, specific to arm1136jf-s. I changed in sources for add more frequencies to cpu and configured to add features. The kernel can be used on any rom.

kernel features:

  • vfat, ext2, ext3, ext4.
  • swap and ramzswap.
  • cpu freq 156-312-624-832
  • cpu auto undervolt on change cpu freq.
  • cpu governors and schedulers.
  • iptables full needed rules.
  • vpn support
  • cifs support
  • init.d - if not work check if the su and busybox location is /system/xbin/
  • suport for bootanimation
  • ClockWorkMOD Recovery v5.0.2.6

How to Compile:

Install dependencies

  1. $ ./install_dep.sh

Configure and Compile

  1. $ ./Compile.sh

Create package

  1. $ ./Creat_package.sh

Samsung Galaxy ACE Model: GT-S5830C-i-M CPU: Broadcom 21553 ARM11 Arquitecture: ARM1136JF-S

This Kernel version 2.6.35.7 is compiled from Samsung sources, with GCC 4.4.7 compiled from sources, specific to arm1136jf-s. I changed in sources for add more frequencies to cpu and configured to add features. The kernel can be used on any rom.

Install dependencies

$ ./install_dep.sh

Configure and Compile

$ ./Compile.sh

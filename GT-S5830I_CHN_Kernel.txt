HOW TO BUILD KERNEL 2.6.35 FOR GT-S5830i

1. How to Build
	- get Toolchain
	Visit http://www.codesourcery.com/, download and install Sourcery G++ Lite 2009q3-68 toolchain for ARM EABI.
	Extract kernel source and move into the top directory.
	$ cd kernel/common/
	$ make ARCH=arm bcm21553_cooperve_defconfig
	$ make ARCH=arm CROSS_COMPILE=../../prebuilt/linux-x86/toolchain/arm-eabi-4.4.3/bin/arm-eabi- modules zImage -j`grep processor /proc/cpuinfo | wc -l`

2. Output files
	- Kernel : kernel/common/arch/arm/boot/zImage
	
3. How to make .tar binary for downloading into target.
	- change current directory to kernel/common/arch/arm/boot
	- type following command
	$ tar cvf GT-S5830i_Kernel_Gingerbread.tar zImage
